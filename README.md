# 📊 IBM Business Intelligence Capstone Project

This project is part of the IBM Business Intelligence Professional Certificate. It demonstrates an end-to-end BI workflow, from data cleaning and SQL analysis to dashboard creation and business reporting.

---

## 🚀 Overview

The objective of this project is to analyze business data and generate actionable insights using multiple tools and techniques across the BI workflow.

The project covers:

- Data cleaning and preparation
- SQL-based data analysis
- Data visualization and statistical analysis
- Dashboard development
- Business presentation of insights

---

## 🛠️ Tools Used

- Excel (Data Cleaning & Analysis)
- PostgreSQL (SQL Queries)
- Tableau (Dashboards)
- PowerPoint (Presentation & Storytelling)

---

## 📌 Project Workflow

### 1. Data Cleaning & Preparation (Excel)

- Cleaned raw dataset
- Handled missing values and inconsistencies
- Standardized data for analysis

---

### 2. Data Querying & Analysis (SQL)

- Wrote SQL queries to extract insights
- Performed aggregations and filtering
- Analyzed trends and relationships in the data

---

### 3. Data Visualization & Statistical Analysis (Excel)

- Created charts and summary tables
- Performed basic statistical analysis
- Identified patterns and trends

---

### 4. Dashboard Development (Tableau)

- Built interactive dashboards
- Designed KPI visualizations
- Enabled filtering and drill-down analysis

---

### 5. Business Presentation (PowerPoint)

- Presented findings in a structured format
- Highlighted key insights and recommendations
- Translated data into business decisions
